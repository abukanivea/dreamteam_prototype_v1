# DreamTeam Candidate Screening Prototype (v1)

This repository supports my selection app, DreamTeam. 
The live MVP is built in Glide and demonstrates a minimal end-to-end hiring flow 
for the Program Aide role at DreamTeam Wellness.

## Live MVP (Glide App)
https://dreamteam-pilot-app-7s4r.glide.page 
## Features Implemented (v1)
- Public job listings page
- Public Application Form (no login required)
- Coordinator Dashboard (hidden from public navigation) with a functional Shortlist toggle
- CEO Review view showing only shortlisted applicants
- CEO “Approve Finalist” button that updates the applicant’s finalDecision status
- Optional demographics (gender, ethnicity) added for fairness monitoring

## Tech Stack
- Glide (no-code platform)
- Glide Tables / Google Sheets (data layer)
- GitHub (documentation repository)

## Setup / Usage
No installation required.
Open the Live MVP link above to test the prototype.
Screens and flows are demonstrated in the Milestone #3 document.
